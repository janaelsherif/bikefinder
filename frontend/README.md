# Frontend (Next.js)

This folder is the **EuropeBikeFinder** web app. **Setup, deploy, and scope** for the whole monorepo are in the **[repository README](../README.md)** and **[DEPLOY.md](../DEPLOY.md)**.

Quick local start (from this directory):

```bash
cp .env.local.example .env.local   # once
npm install
npm run dev
```

Open **http://localhost:3000** (locale routes: **`/de-CH/…`**, **`/en/…`**).
